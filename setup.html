Setup your computer

This page shows how to setup your computer ready to plug in and program the Wemos D1 R2.

The Wemos is basically the brain of your hedgehog bot, connecting it to the internet, telling the motors to move, the screen what to display.

You only need to follow these instructions if you do NOT have the workshop memory stick!

1. Download the arduino software

  1. Visit arduino.cc and download the software for your computer (Windows, Mac OSX, Linux..)
  2. Open the installer and install it. (Click: I Agree, Next, Install, Close)
  3. Open Arduino - there should be a shortcut on your desktop or start menu / dock.

2. Setup arduino software for WeMos

  1. In Arduino, click File > Preferences.
    • Copy http://arduino.esp8266.com/stable/package_esp8266com_index.json
    • paste it in the Additional Boards Manager URLs box.
    • Click OK.

  2. Click Tools > Board:"..." > Boards Manager....
    • Click Type > Contributed.
    • Click on esp8266 by ESP8266 Community and click Install.
  3. Click Tools > Board:"..." > Wemos D1 R2 & Mini.

3. Add Libraries to arduino

  1. Download the Hedgehog Bot Libraries from here.
  2. Click on Libraries.zip and the download button in the top centre.
  3. Extract the zip file.
  4. Copy the folders inside to your arduino libraries folder (Documents/Arduino/libraries)

    Setup Complete!

    As you are all set up, you can skip the Setup section of Challenge 1.
